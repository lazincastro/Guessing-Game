## ğŸš€ How It Works

The game chooses a secret number within a range based on the selected difficulty level.

You have a limited number of attempts to guess the correct number.

After each guess, the game tells you whether the secret number is:

- Greater than your guess
- Less than your guess

Your score decreases based on how far your guess is from the correct number.
